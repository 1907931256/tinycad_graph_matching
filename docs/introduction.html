An Introduction to TinyCAD

 

Welcome to TinyCAD!

You can get the latest news, release and source code from the Sourceforge site. Go to http://tinycad.sourceforge.net.

TinyCAD is an open source schematic capture program. Use TinyCAD to draw your own circuit diagrams.

You can lay out circuit diagrams using the supplied libraries. Alternatively, there is a component library editor built into TinyCAD – so you can easily create your own symbols. You may also find more libraries on the Yahoo group that other users have drawn for you.

Once you have drawn your design, you can use the special functions to automatically add symbol references, check your design against rules of correctness and create a parts list. TinyCAD can also export a netlist for import into a PCB layout program.

Once you have your design you can copy and paste it into other packages such as Word (or better still OpenOffice). You may also print your design to any supported Windows printer. If your design is larger than a single page then it can be either scaled to fit on a single page or printed across many pages.

Changes since version 1.80.00

Welcome to TinyCAD version 1.90.00. Here is a quick summary that has changed in this release:

New symbol picker tool
The most obvious change to TinyCAD when you fire it up is the new symbol picker that sits on the left hand side of the screen. This replaces the old way of fetching symbols from the libraries.

New library format
The Libraries are now stored in Microsoft Access database format. This means all new libraries will have an extension of .mdb and can be opened by a copy of Access. This change to a more standard format for storing libraries will make it much easier to add new features to the libraries in future, and will enable 3rd parties to write their own tools to read/modify the symbols.

The first new feature with the new library format, is that you are able to select the default orientation of symbols when they are picked. This fixes a long running bug in TinyCAD that symbols were drawn one way round, but always extracted from the libraries rotated.

To convert your libraries to the new format, use the new “Upgrade” button in the library dialogue box.

Lots of bug fixes
In line with all new releases I have been busy fixing bugs. There are a few crashes fixed and some tiding up of the way editing drawings work.
